This repository contains my data analysis practice projects using the Python **Pandas** library.

* **Data Cleaning Practice**: Handled missing values, dropped duplicates, and changed data types.

## 🚀 Key Libraries Used
* Pandas
* NumPy
* Matplotlib (for data visualization)
