# Pandas Data Analysis Practice 📊

This repository contains my data analysis practice projects using the Python **Pandas** library.

## 💡 Project Attribution & Credits
The data and core challenges in this repository come from [Keith Galli's Pandas Data Science Tasks (https://github.com/KeithGalli/Pandas-Data-Science-Tasks).

* **My Implementation**: While following the project objectives, my code and problem-solving approaches are a bit different from Keith Galli's original solutions. I have customized the workflows, logic, and optimization steps to better fit my own learning style and coding preferences.

## 📁 Projects Included
* **Sales Data Analysis**: Merging 12 months of sales data into a single CSV, cleaning the dataset, and performing exploratory data analysis (EDA).
* **Business Insights**: Finding the best sales months, identifying peak advertising hours, and discovering which products are most frequently sold together.

## 🚀 Key Libraries Used
* Pandas
* Matplotlib (for data visualization)
